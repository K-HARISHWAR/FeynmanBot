# FeynmanBot - Architecture

## Frontend
- **React & Vite**: Fast development and rendering.
- **Tailwind CSS**: Utility-first styling for a clean, modern SaaS look.
- **React Router**: For seamless single-page application navigation.

## Backend
- **FastAPI**: High-performance Python framework.
- **Pydantic**: Data validation and serialization.
- **AI Service**: Modular integration with Gemini/OpenAI for generating follow-up questions and final evaluation reports.

## Database
- **Supabase PostgreSQL**: Scalable relational database storing user profiles, teaching sessions, Q&A history, and final evaluation reports.
